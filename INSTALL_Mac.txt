
This binary distribution for macOS 10.14 64bit Intel is the contribution from Sjoerd W. Bijleveld.

Note: This is a "beta" release because does not correctly work in some DAWs and Digital Performer could not load this release.


Installation

(1) Unzip the file. Then you will see the following two FOLDERS.
  SoundObject.vst	contains SoundObject plug-in.
  suzumushi		contains a few VST preset files.

(2) Copy SoundObject.vst3 to /Library/Audio/Plug-ins/VST3

Note: "SoundObject.vst3" is a FOLDER in the unzipped file.
It is NOT "SoundObject.vst3/Contents/MacOS/SoundObject".

(3) Copy suzumushi to /Users/your_account_name/Library/Audio/Presets

Enjoy.
