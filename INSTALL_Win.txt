Installation

(1) Unzip the file. Then you will see the following two FOLDERS.
  SoundObject.vst	contains SoundObject plug-in.
  suzumushi		contains a few VST preset files.

(2) Copy SoundObject.vst3 to C:\Program Files\Common Files\VST3

Note: "SoundObject.vst3" is a FOLDER in the unzipped file.
It is NOT "SoundObject.vst3\Contents\x86_64-win\SoundObject.vst3".

(3) Copy suzumushi to C:\Users\your_account_name\Documents\VST3 Presets

Enjoy.


インストール方法

(1) ファイルを解凍すると，以下の 2 つの「フォルダ」がある．
  SoundObject.vst	SoundObject プラグイン
  suzumushi		VST preset ファイル

(2) SoundObject.vst3 を C:\Program Files\Common Files\VST3 にコピー

Note: "SoundObject.vst3" は解凍したファイルにある「フォルダ」です．
"SoundObject.vst3\Contents\x86_64-win\SoundObject.vst3" ではありません．

(3) suzumushi を C:\Users\your_account_name\Documents\VST3 Presets にコピ
ー

Enjoy.
